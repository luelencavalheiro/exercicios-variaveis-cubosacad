## Fahrenheit x Celsius

Para convertermos graus em Fahrenheit para graus em Celsius utilizamos a seguinte fórmula:

$$ C = (F - 32) \* \frac{5}{9} $$

Onde,

**F:** grau em Fahrenheit | **C:** grau em Celsius
